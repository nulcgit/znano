# Znano

Znano - distributed recommendation system / Знано - распределенная рекомендательная система

### Requirements

x86_64/ARM64; Ubuntu 24.04; 1 Core; 2GB RAM; 20GB SSD; 20mbps WAN; Open Ports TCP+UDP: 4011

### Installation

1. `git clone https://github.com/nulcgit/znano.git znano && cd znano && bash install.sh`
2. System will reboot.
